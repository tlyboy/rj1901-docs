> 在使用计算机的过程中，经常会碰到一些体积比较大的文件或者是比较零碎的文件，这些文件放在计算机中会占据比较大的空间，也不利于计算机中文件的整理。此时可以使用 WinRAR 将这些文件压缩，以便管理和查看。
